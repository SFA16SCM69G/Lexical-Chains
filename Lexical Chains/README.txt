lexicalChain.py implements the noun lexical chains
Run Step:
1.put the lexicalChain.py and test files at the same directory
2.python lexicalChain.py filename[test.txt]

summarize.py implements a text summarizer
Run Step:
1.put the summarize.py and test files at the same directory
2.python summarize.py filename[test.txt] ratio[0<=ratio<=1]

The second parameter ratio = the number of sentences output/the number of all sentences
The order of output is based on the weight of each sentences.
The detail determination of weight is shown on my report.
